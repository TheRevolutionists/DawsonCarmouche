# Dawson Carmouche Personal Portfolio Website

A modern, interactive, and aesthetic personal website for Dawson Carmouche, featuring a black, white, and purple color scheme.

## Features
- Responsive design with burger menu navigation
- Home, About, Portfolio, Skills, Resume, Contact, and Blog pages
- Contact form (mailto)
- Downloadable resume
- Modern, eye-catching style

## Setup
1. Clone or download this repository.
2. Open `index.html` in your browser to view the website.
3. To customize, edit the HTML files and `styles.css` as needed.
4. Add your resume PDF as `resume.pdf` in the root directory for the download link to work.

## Customization
- Update project cards in `portfolio.html` with your real projects.
- Add your resume content or embed a PDF in `resume.html`.
- Edit skills in `skills.html`.
- Add blog posts in `blog.html` when ready.
